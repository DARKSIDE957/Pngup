## What it does

Pngup is a lightweight always-on-top photo overlay for Windows. Drop in an image, position it anywhere on screen, and use your apps underneath when you need to — without closing the photo.

Perfect for reference images, design mockups, stream layouts, study notes, or keeping inspiration visible while you work.

## Features

| | |
| --- | --- |
| **Floating photo** | Transparent, always-on-top window that stays out of your way |
| **Click-through** | Let mouse clicks pass through the photo when you want to work behind it |
| **Draggable controls** | Settings ball you can place anywhere on your monitor |
| **Resize & opacity** | Scale the photo and dial transparency to taste |
| **Themes** | Dark and light UI |
| **6 languages** | English, Arabic (RTL), Spanish, French, German, Russian |
| **System tray** | Quick actions from the notification area |
| **Remembers settings** | Position, size, theme, and language persist across restarts |

## Get started

1. **Download** the installer from [Releases](https://github.com/DARKSIDE957/Pngup/releases).
2. **Run** `Pngup_0.1.1_x64-setup.exe` and finish setup.
3. **Pick a photo** from the settings ball or tray menu.
4. **Drag** the photo to position it. Toggle **Pass clicks through** when you want to use apps behind it.

## Keyboard shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl+O` | Pick photo |
| `Ctrl+T` | Toggle pass clicks through |
| `Ctrl+Shift+H` | Show settings |
| `Ctrl+Shift+P` | Toggle pass clicks through (global) |
| `Ctrl+Shift+O` | Pick photo (global) |

## Tray menu

Right-click the tray icon for quick access to:

- Show photo
- Show settings
- Pick photo
- Toggle pass clicks through
- Toggle theme
- Quit
